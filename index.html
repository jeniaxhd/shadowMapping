01

Stiahnuť LWJGL a JOML

Stiahni lwjgl-3.3.4.zip z github.com/LWJGL/lwjgl3/releases/tag/3.3.4 a rozbaľ kamkoľvek na disk. JOML nie je súčasťou archívu — stiahni joml-1.10.7.jar osobitne z mvnrepository.com (klikni jar pri Files).

02

Nový projekt v IntelliJ IDEA

New Project → Java → Build system: IntelliJ → JDK 21 → Create.

03

Pridať LWJGL do Project Structure

File → Project Structure → Libraries → + → Java → naviguj do rozbaleného archívu a vyber celé tieto priečinky naraz (Ctrl+klik). IDEA pridá všetky .jar v nich automaticky.

vybrať tieto priečinky
lwjgl/ lwjgl-opengl/ lwjgl-glfw/ lwjgl-stb/

Potom pridaj ešte joml-1.10.7.jar rovnako cez + → Java. Klikni OK → Apply → OK.

04

Overiť — spustiť testovací súbor

Vytvor Test.java, skopíruj kód a spusti cez ▶. Otvorí sa čierne okno 800×600 — ak ho vidíš, všetko funguje.

Test.java
import org.lwjgl.glfw.GLFW; import org.lwjgl.opengl.*; import static org.lwjgl.glfw.GLFW.*; import static org.lwjgl.opengl.GL11.*; public class Test { public static void main(String[] args) { glfwInit(); long win = glfwCreateWindow(800, 600, "LWJGL OK", 0, 0); glfwMakeContextCurrent(win); GL.createCapabilities(); System.out.println("OpenGL: " + glGetString(GL_VERSION)); while (!glfwWindowShouldClose(win)) { glClear(GL_COLOR_BUFFER_BIT); glfwSwapBuffers(win); glfwPollEvents(); } glfwTerminate(); } }

V konzole musíš vidieť OpenGL: 4.x.x. Ak je verzia nižšia ako 3.3 alebo program padne — kontaktuj lektora pred workshopom.

Princíp shadow mappingu teória
Dva renderovacie prechody. Depth map z pohľadu svetla. Light space matrix. Shadow acne a bias.

Nastaviť FBO pomocou glGenFramebuffers, napísať depth shader, vizualizovať depth textúru.
Shadow test teória
Porovnanie hĺbky vo fragment shaderi. Bias trick — shadow acne vs peter panning.

Zapojiť depth map do hlavného shadera, napísať shadow test, doladiť bias.
Bonusové cvičenie — PCF bonus
